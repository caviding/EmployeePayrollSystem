# Employee Payroll System

A **Java-based Employee Payroll System** built using **JDBC** and **MySQL**.
This project demonstrates how to manage employees and payroll data using core Java concepts, database connectivity, and SQL operations.

The project is suitable for **junior Java developers** who want practical experience with JDBC and relational databases.
